── IMPORTANT: Dataset Instructions ──────────────────────────

1. Download dataset from Kaggle:
   https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

2. Extract the ZIP and place files here:
   FakeNewsProject/
   └── dataset/
       ├── Fake.csv
       └── True.csv

3. Then run:  python train_model.py
   This will create model/model.pkl and model/vectorizer.pkl automatically.
